二、centos7
1.centos7换源
    1) sudo cp /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup
    2) wget -O /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo
    3) yum makecache
    4) sudo yum update
2.centos7更换主机名
    1)查看当前主机名
        hostnamectl
    2)修改主机名
        永久修改
            1.hostnamectl set-hostname 新主机名
            2.编辑/etc/hostname文件来修改主机名
                vim /etc/hostname -> 将文件内容修改为新的主机名 -> 重启
        临时修改
            hostname 新主机名
            
三、Git配置 
1.设置用户名和邮件地址
    git config --global user.name "John"
    git config --global user.email johndoe@example.com
2.设置代理
    git config --global http.proxy http://127.0.0.1:7890
    git config --global https.proxy http://127.0.0.1:7890
3.配置使用初始分支名
    git config --global init.defaultBranch <名称>
4.使用ssh连接github
    1)检查并生成密钥
        ls ~/.ssh 查看是否已有ssh密钥
        su
        ssh-keygen -t rsa -C "邮箱"
    2)复制公钥并添加至远程仓库
        cat ~/.ssh/id_rsa.pub
        登录github -> setting -> ssh and GPG keys -> new ssh key，粘贴ssh公钥
    3)测试ssh连接
        ssh -T git@github.com
    4)克隆仓库
        git clone git@github.com:用户名/仓库名.git
