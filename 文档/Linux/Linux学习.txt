Vim游戏：https://github.com/jmoon018/PacVim.git
    安装方式:
        1.Download and install Curses (graphics library)
            -> For Ubuntu (in terminal): sudo apt-get install libncurses5-dev
        2.git clone https://github.com/jmoon018/PacVim.git
        3.cd PacVim
        4.sudo make install
    启动方式：
        pacvim [LEVEL_NUMER] [MODE]
        [LEVEL_NUMER]:表示关卡号
        [MODE]:表示难度，n - 正式模式，h - 困难模式
        例：$ pacvim 5 n    #表示玩第 5 关，正式模式
            $ pacvim        #表示从头开始玩游戏
    游戏规则：
        基本规则：
            1.操纵主角，躲避怪物，吃完豆豆。
                主角：绿色方块（终端光标，如设置成下划线则为绿色下划线）
                怪物：红色大写字母G
                豆豆：白色单词，在主角经过时，字符会变成绿色
            2.主角共有 3 条命，在通过第 0，3，6，9 关时，会额外奖励一条生命。
            3.游戏一共 10 关，从 0 ~ 9，通过第 9 关后，将重新从第 0 关开始，但难度加大。
            4.游戏内有两种障碍物：
                黄颜色的是代表墙壁，你不能越过墙壁；
                青色的飘号键 ~ ，它不会攻击你，但你也不能碰它，一碰你就挂。
        胜利的条件：
            操作绿色方块，将所有的白色字符都变成绿色时，就赢了那一关，进入到下一关。
        失败的条件：
            当碰到了怪物（大写字母 G）或飘号键 ~ ，就失去一条生命。所有生命都失去时，游戏结束。    
    操作方式：
        q	    退出游戏    
        h	    左移    
        j	    下移    
        k	    上移    
        l	    右移    
        w	    前进到下一个小写单词的开头
        W	    前进到上一个大写单词的开头
        e	    前进到下一个小写单词的结尾
        E	    前进到上一个大写单词的结尾
        b	    后移到下一个小写单词的开头
        B	    后移到下一个小写单词的开头
        $	    移动到行尾
        0	    移动到行首
        gg/1G	移动到第一行的开头
        nG	    移动到第n行的开头
        G	    移动到最后一行的开头
        ^	    移动到当前行的第一个单词
        &	    1337 cheatz（击败当前水平）
